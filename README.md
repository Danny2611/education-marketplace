# Education Marketplace - Frontend

Sàn giáo dục thương mại điện tử tích hợp AI

## Features

- 🔍 Tìm kiếm và lọc sản phẩm
- 🤖 Gợi ý thông minh AI
- ❤️ Yêu thích sản phẩm
- 📱 Responsive design
- 💬 Chatbot tư vấn
- 📊 Lịch sử xem

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Lucide React Icons

## Installation

```bash
npm install
npm start
```
